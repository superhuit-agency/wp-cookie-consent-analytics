=== Description ===

Cookie Consent Analytics empowers you to track user interactions with your cookie banner through dedicated API routes that record acceptances, rejections, and personalization choices. 
The plugin includes a user-friendly dashboard to visualize analytics with interactive graphs and enables you to easily export your data in CSV format for further analysis.
